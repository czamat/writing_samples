How to install the samples?

Last Updated: 2021-03-01

Samples installation in IBM Cognos Analytics version 11.0.x is different than in IBM Cognos Business Intelligence (BI) version 10.2.2. There are new samples that were created specifically for Cognos Analytics version 11.0.x, but the samples from previous Cognos BI releases can also be used.

The installation process is slightly different for different groups of samples.

  • Base samples
    These samples are installed by default when you perform an Easy install of IBM Cognos Analytics server, or when you select the samples check box in a Custom install. The samples are installed to the cognos_analytics_server_install_location\samples folder.
    Tip: The samples option is not available with the Cognos Analytics AIX installation kit, version 11.0.7 and later. For more information, see this article.
  • Extended samples

    These samples have their own install kit bisamples, and must be installed to a different folder than Cognos Analytics.

  • Supplementary samples

    These samples are available from Supplementary (Legacy) IBM Cognos Analytics 11 Samples (www.ibm.com/communities/analytics/cognos-analytics-blog/supplementary-ibm-cognos-analytics-11-samples).

For more information, see the Samples for IBM Cognos Analytics Guide.
