Installing BI Samples

Samples is a separate installation from Cognos Analytics Server.

The following topics explain how to install Cognos Analytics Samples. After you have installed Cognos Analytics Samples, the samples can be found in cognos_samples_installation_locationwebcontent/samples. This folder has the following sub-folders.

content

This folder contains the sample deployment archives.

datasources

This folder contains the sample databases.

models

This folder contains the sample IBM Cognos Framework Manager models.
